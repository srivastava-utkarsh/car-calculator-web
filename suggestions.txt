To achieve **left and right panels each occupying exactly 30% of the container width**, you should use Tailwind’s utility classes with either [arbitrary values](https://tailwindcss.com/docs/width#arbitrary-values) for width or, for grid layouts, use the `basis-*`, `w-*`, or custom grid column configurations.

### **Recommended Config for 30%/40%/30% Layout (Left/Middle/Right)**

If you want **left 30%**, **right 30%**, and optionally a center “main” region with the remaining 40%, here’s how you can do it:

#### **Option 1: Flexbox with Width Fraction Utilities (Easiest for Percentage)**
Use `flex` and set each part’s width individually:

```html

  
    
      
      
        Left (30%)
      
      
      
        Main (40%)
      
      
      
        Right (30%)
      
    
  

```
- `w-[30%]` assigns 30% width (arbitrary Tailwind width).
- `flex-1` fills remaining space (40% if used).

#### **Option 2: CSS Grid with Custom Column Template**
Let’s say you want exactly three columns, with col1/col3 at 30% and the middle at 40%:

```html

  
    
      
      
        Left (30%)
      
      
      
        Main (40%)
      
      
      
        Right (30%)
      
    
  

```

### **Key Point:**
- For **custom widths like 30%**, use Tailwind’s arbitrary value syntax:  
  - `w-[30%]` (for flex children)  
  - `[grid-template-columns:30%_1fr_30%]` (for grid container)
